# Lava Test Apps

Example applications for testing Lava Monetize integration.

## Apps

| App | Description |
|-----|-------------|
| [travel-advisor](./travel-advisor) | AI travel advisor demo with checkout flow and usage-based billing |

## Adding a New Test App

1. Create a folder for your app
2. Include a README with setup instructions
3. Remove any secrets/PII - use `.env.example` files
4. Submit a PR
